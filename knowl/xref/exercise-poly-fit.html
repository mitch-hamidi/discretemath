Exercise 9.

In a previous math class, you have probably seen the fact that, if we are given two points in the plane, then there is a unique line passing through both of them. In this problem, we will begin with the four points on the left below and ask to find a polynomial that passes through these four points as shown on the right.
A degree three polynomial can be written as
\begin{equation*} p(x) = a + bx + cx^2 + dx^3 \end{equation*}
where \(a\text{,}\) \(b\text{,}\) \(c\text{,}\) and \(d\) are coefficients that we would like to determine. Since we want the polynomial to pass through the point \((3,1)\text{,}\) we should require that
\begin{equation*} p(3) = a + 3b + 9c + 27d = 1\text{.} \end{equation*}
In this way, we obtain a linear equation for the coefficients \(a\text{,}\) \(b\text{,}\) \(c\text{,}\) and \(d\text{.}\)

  2. Write the augmented matrix corresponding to this system of equations and use the Sage cell below to solve for the coefficients.
  3. Write the polynomial \(p(x)\) that you found and check your work by graphing it in the Sage cell below and verifying that it passes through the four points. To plot a function over a range, you may use a command like plot(1 + x- 2*x^2, xmin = -1, xmax = 4).
  4. Rather than looking for a degree three polynomial, suppose we wanted to find a polynomial that passes through the four points and that has degree two, such as
    \begin{equation*} p(x) = a + bx + cx^2\text{.} \end{equation*}
    Solve the linear system for the coefficients. What can you say about the existence and uniqueness of a degree two polynomial passing through these four points?
  5. Rather than looking for a degree three polynomial, suppose we wanted to find a polynomial that passes through the four points and that has degree four, such as
    \begin{equation*} p(x) = a + bx + cx^2 + dx^3 + ex^4\text{.} \end{equation*}
    Solve the linear system for the coefficients. What can you say about the existence and uniqueness of a degree four polynomial passing through these four points?
  6. Suppose you had 10 points and you wanted to find a polynomial passing through each of them. What should the degree of the polynomial be to guarantee that there is exactly one such polynomial? Explain your response.
Answer.
  1. The four equations we find are
    \begin{equation*} \begin{alignedat}{5} a \amp \amp \amp \amp \amp \amp \amp {}={} \amp 2 \\ a \amp {}+{} \amp b \amp {}+{} \amp c \amp {}+{} \amp d \amp {}={} \amp 3 \\ a \amp {}+{} \amp 2b \amp {}+{} \amp 4c \amp {}+{} \amp 8d \amp {}={} \amp 0 \\ a \amp {}+{} \amp 3b \amp {}+{} \amp 9c \amp {}+{} \amp 27d \amp {}={} \amp 1\text{.} \\ \end{alignedat} \end{equation*}
  2. The coefficients are \(a=2\text{,}\) \(b=\frac{17}{3}\text{,}\) \(c=-6\text{,}\) and \(d=\frac43\text{.}\)
  3. The polynomial is
    \begin{equation*} p(x) = 2 + \frac{17}{3}x -6x^2 + \frac43 x^3\text{.} \end{equation*}
  4. The linear system is inconsistent.
  5. The linear system is consistent with infinitely many solutions.
  6. Nine.
Solution.
  1. The four equations we find are
    \begin{equation*} \begin{alignedat}{5} a \amp \amp \amp \amp \amp \amp \amp {}={} \amp 2 \\ a \amp {}+{} \amp b \amp {}+{} \amp c \amp {}+{} \amp d \amp {}={} \amp 3 \\ a \amp {}+{} \amp 2b \amp {}+{} \amp 4c \amp {}+{} \amp 8d \amp {}={} \amp 0 \\ a \amp {}+{} \amp 3b \amp {}+{} \amp 9c \amp {}+{} \amp 27d \amp {}={} \amp 1\text{.} \\ \end{alignedat} \end{equation*}
  2. This leads to the augmented matrix
    \begin{equation*} \left[\begin{array}{rrrr|r} 1 \amp 0 \amp 0 \amp 0 \amp 2 \\ 1 \amp 1 \amp 1 \amp 1 \amp 3 \\ 1 \amp 2 \amp 4 \amp 8 \amp 0 \\ 1 \amp 3 \amp 9 \amp 27 \amp 1 \\ \end{array}\right] \sim \left[\begin{array}{rrrr|r} 1 \amp 0 \amp 0 \amp 0 \amp 2 \\ 0 \amp 1 \amp 0 \amp 0 \amp \frac{17}{3} \\ 0 \amp 0 \amp 1 \amp 0 \amp -6 \\ 0 \amp 0 \amp 0 \amp 1 \amp \frac43 \\ \end{array}\right]\text{.} \end{equation*}
    This gives the coefficients \(a=2\text{,}\) \(b=\frac{17}{3}\text{,}\) \(c=-6\text{,}\) and \(d=\frac43\text{.}\)
  3. The polynomial
    \begin{equation*} p(x) = 2 + \frac{17}{3}x -6x^2 + \frac43 x^3 \end{equation*}
    passes through the four given points.
  4. We have the augmented matrix
    \begin{equation*} \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 2 \\ 1 \amp 1 \amp 1 \amp 3 \\ 1 \amp 2 \amp 4 \amp 0 \\ 1 \amp 3 \amp 9 \amp 1 \\ \end{array}\right] \sim \left[\begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \\ \end{array}\right]\text{.} \end{equation*}
    This system is therefore inconsistent, which means there is no degree 2 polynomial passing through the four points.
  5. The augmented matrix we find is
    \begin{equation*} \left[\begin{array}{rrrrr|r} 1 \amp 0 \amp 0 \amp 0 \amp 0 \amp 2 \\ 1 \amp 1 \amp 1 \amp 1 \amp 1 \amp 3 \\ 1 \amp 2 \amp 4 \amp 8 \amp 16 \amp 0 \\ 1 \amp 3 \amp 9 \amp 27 \amp 81 \amp 1 \\ \end{array}\right] \sim \left[\begin{array}{rrrrr|r} 1 \amp 0 \amp 0 \amp 0 \amp 0 \amp 2 \\ 0 \amp 1 \amp 0 \amp 0 \amp 6 \amp \frac{17}{3} \\ 0 \amp 0 \amp 1 \amp 0 \amp -11 \amp -6 \\ 0 \amp 0 \amp 0 \amp 1 \amp 6 \amp \frac43 \\ \end{array}\right]\text{.} \end{equation*}
    This shows that there are infinitely many polynomials whose degree is four and that pass through the four points.
  6. If there are 10 points, we should find a polynomial having 10 coefficients. This means that the degree of the polynomial should be 9.
