Exercise 12.

We defined an \(n\times n\) matrix to be invertible if there is a matrix \(B\) such that \(AB=I_n\text{.}\) In this exercise, we will explain why it is also true that \(BA = I\text{,}\) which is the statement of PropositionÂ 3.1.6. This means that, if \(B=A^{-1}\text{,}\) then \(A = B^{-1}\text{.}\)
  1. Suppose that \(\xvec\) is an \(n\)-dimensional vector. Since \(AB=I\text{,}\) explain why \(AB\xvec = \xvec\) and use this to explain why the only vector for which \(B\xvec = \zerovec\) is \(\xvec = \zerovec\text{.}\)
  2. Explain why this implies that \(B\) must be invertible. We will call the inverse \(C\) so that \(BC = I\text{.}\)
  3. Beginning with \(AB = I\text{,}\) explain why \(B(AB)C = BIC\) and why this tells us that \(BA = I\text{.}\)
Answer.
  1. If \(B\xvec = \zerovec\text{,}\) then \(AB\xvec = \xvec = \zerovec\text{.}\)
  2. \(B\) is invertible because \(B\sim I\text{.}\)
  3. Multiply \(AB=I\) on the left by \(B\) and the right by \(C\) and regroup the matrix multiplications.
Solution.
  1. If \(B\xvec = \zerovec\text{,}\) then \(AB\xvec = A\zerovec = \zerovec = \xvec\text{.}\)
  2. Since the only solution to the homogeneous equation \(B\xvec = \zerovec\) is \(\xvec=\zerovec\text{,}\) we know that the columns of \(B\) are linearly independent and so \(B\) has a pivot position in every column. Since \(B\) is a square matrix, this tells us that \(B\sim I\) so that \(B\) is invertible.
  3. If we multiply \(AB=I\) on the left by \(B\) and the right by \(C\text{,}\) then we have
    \begin{equation*} \begin{aligned} B(AB)C \amp = BIC \\ BA(BC) \amp = BC \\ BAI \amp = I \\ BA \amp = I \end{aligned}. \end{equation*}
