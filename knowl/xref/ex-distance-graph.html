Example 9.5.2. A Distance Graph.

Let \(V\) be the set of six capital cities in New England: Boston, Augusta, Hartford, Providence, Concord, and Montpelier. Let \(E\) be the set of all possible undirected edges between these cities. So \((V,E)\) is a complete unordered graph. An example of a weight function on this graph is \(w\left(\{c_1, c _2\right\}) = \textrm{the distance, in miles, between } c_1 \textrm{ and } c_2\text{.}\)
